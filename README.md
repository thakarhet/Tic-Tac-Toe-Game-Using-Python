# Tic-Tac-Toe-Game-Using-Python

# ❌⭕ Tic Tac Toe

A simple **command-line Tic Tac Toe** game built using **Python**. Two players take turns placing their marks (`X` and `O`) on a 3×3 board until one player wins or the game ends in a draw. The project demonstrates the fundamentals of Python, including loops, functions, conditionals, lists, exception handling, and game logic.

---

## 🚀 Features

* 🎮 Two-player gameplay
* 🧩 3×3 game board
* ✅ Win detection
* 🤝 Draw detection
* ❌ Input validation
* 🔄 Alternating player turns
* 💻 Simple command-line interface

## 🎮 How to Play

1. The game starts with **Player X**.
2. Players enter a position number between **0** and **8**.
3. The board positions are arranged as follows:

```text
0 | 1 | 2
--|---|--
3 | 4 | 5
--|---|--
6 | 7 | 8
```

4. Players take turns placing their marks.
5. The game ends when:

   * A player gets three marks in a row, column, or diagonal.
   * All positions are filled, resulting in a draw.
